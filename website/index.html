Features

Everything runs on your Mac.

🔒

100% On-Device

WhisperKit runs on the Neural Engine, LLM runs on Metal via MLX. No data ever leaves your Mac.

⌨️

Auto-Paste

Transcribed text is automatically pasted wherever your cursor is. No copy-paste needed.

🧹

LLM Cleanup

Optional local LLM pass to fix punctuation, grammar, and formatting before pasting.

🎯

Context-Aware

Select text before dictating to give the LLM context. Per-app prompts for different workflows.

💬

Floating Indicator

Translucent pill at your text cursor shows recording state and audio level in real time.

⚡

Menu Bar Native

Lives in your menu bar. Configurable hotkey, hold or toggle mode. Stays out of the way.

Installation

Download the latest release, unzip, and drop into Applications. The app is ad-hoc signed (not notarized), so you'll need to clear the quarantine flag on first launch.

Install via Terminal
$ curl -fsSL https://nikoro.github.io/dictum/install.sh | bash
Downloads the latest release, moves to /Applications, and clears quarantine — all in one step.
Prefer to install manually?
1 Download Dictum.zip and extract it
2 Move Dictum.app to /Applications
3 Run in Terminal:
xattr -dr com.apple.quarantine /Applications/Dictum.app
After installing, launch Dictum and follow the onboarding: grant permissions, download a speech model, and optionally an LLM.
